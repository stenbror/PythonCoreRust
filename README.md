# PythonCoreRust

Python 3.10 parser in a Roslyn style thinking

Build with:  cargo build --release
Test with:   cargo test
Run with:    cargo run


Minimalize with:
  cargo build --release
  cd target/release
  strip python_core_rust
  ls -la
